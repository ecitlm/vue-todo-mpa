### vue-todo cli3
>基于vue-cli3脚手架升级封装、可拿来即用、已集成基本常用模块

### start Install
```shell
git clone https://github.com/ecitlm/vue-todo-mpa.git
cd vue-cli3

# use npm
npm install

# or use yarn
yarn install 

#server start
npm run serve
# localhost:9527
```

### used module
1. `vue-router`集成、并模块化抽离‘
2. `axios`网络请求封装、以及拦截器判断
3. `vuex`集成
4. `postcss`集成、`px转rem`
5. `filter`拦截器封装并抽离
6. `sessionStorage`集成
